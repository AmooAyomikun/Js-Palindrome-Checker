**Palindrome Checker App**
The Palindrome Checker App is a lightweight JavaScript web application that determines whether a given word or phrase is a palindrome.
A palindrome is a word, phrase, or sequence that reads the same backward as forward when case, spacing, and non-alphanumeric characters are ignored.
This project demonstrates foundational front-end development concepts including:
  DOM manipulation
  Event-driven programming
  String processing and transformation
  Conditional logic
  UI state management
  Code structuring and separation of concerns
Developed as part of my front-end JavaScript learning journey, focusing on writing clean, structured, and maintainable code.

**Features**
  Input validation with user-friendly error messages
  Automatic removal of spaces, punctuation, and special characters
  Case-insensitive comparison
  Visual success and error state feedback
  Reset functionality
  Error message clears dynamically when typing

**How It Works**
The user enters text into the input field.
On clicking the "Check" button:
The input is trimmed.
Non-alphanumeric characters are removed.
The string is converted to lowercase.
The cleaned string is reversed.
The original cleaned string is compared to the reversed string.

A boolean result (true or false) determines the displayed message.
